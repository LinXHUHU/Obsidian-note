# 2.1 处理器特点
1. 通用处理器GPP：mcu和mpu都包含，但两个添加了自己的特点
2. mpu：通常表示一个cpu
3. mcu：强调cpu 存储外围电路都放在一个芯片![[BSP/asset/Pasted image 20250915164310.png]]
4. 主流CPU核
	1. ARM
	2. MIPS
	3. powerPC
	4. 68K
5. cpu体系架构
	1. 冯诺依曼
	2. 哈佛
6. 指令集角度
	1. RISC简单指令集
	2. CISC复杂指令集
7. 数字信号处理器DSP
	1. 针对图像语音视频算法设计
	2. 有独立的乘法器，乘法卷积，滤波等算法计算更快
	3. 采用改进的哈佛架构![[BSP/asset/Pasted image 20250915165025.png]] 有独立的地址总线和数据总线，两个总线有程序存储器和数据存储器分时共用
	4. 定点DSP和浮点DSP： 浮点运算有硬件实现单周期完成，顶点DSP只能模拟
	5. [[BSP/GPP和DSP两者的区别]] 界限逐渐模糊
	6. **射频 = 可用于无线通信的高频电信号**
8. 专用处理器ASP
9. 网络处理器
10. ![[BSP/asset/Pasted image 20250915183702.png]]

# 2.2 存储器介绍
1. 存储器分类
	1. NVM
		1. ROM
			1. 不可变成ROM
			2. PROM
			3. EPROM
			4. EEPROM ， 完全可以用软件擦写
		2. Flash
			1. NOR
				1. cpu接口类似SRAM
				2. 可芯片内执行代码
				3. [[为什么nor可以芯片内执行代码]]
					1. 数据地址总线
					2. 随机访问
					3. 读取速度快（写入擦除慢）
				4. ![[BSP/asset/Pasted image 20250915191221.png]]
			2. NAND
				1. [[NAND锁存的作用]]
				2. 容易错，使用ECC算法
				3. 所有flash的初始值都是ff，擦除回复，写1编0
			3. [[BSP/nor flash 和 nand flash 两者区别]]
			4. rom 有被flash替代的趋势
			5. CFI 公共闪存接口，可以随时替换旧的存储器
			6. 光介质，磁介质
	2. RAM
		1. SRAM
		2. DRAM
		3. [[DRAM和SRAM区别]]
