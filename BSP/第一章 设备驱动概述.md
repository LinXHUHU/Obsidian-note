
# 概念作用
 1. 驱动就是软件和硬件之间的纽带
 2. 软件逻辑不能直接操作硬件（可重用性）
 3. 高内聚低耦合
# 无操作系统
1. 死循环中遍历处理中断
2. 链接软件和硬件的桥梁 ^f32f8f
# 有操作系统
1. 设备驱动融入内核
2. 需要面向操作系统内核接口开发
3. 链接硬件和内核的桥梁 
4. 操作系统给驱动开发制造困难达到给软件提供便利
# 驱动和系统软硬件关系
1. 设备分类
	1. 字符设备：必须串行访问
	2. 块设备：任意顺序访问
	3. 网络设备：面向数据包的接收发送
	4. 其他独特的驱动：tty驱动，iic驱动，usb
